FAQ FoliOpus

Apakah saya harus jago coding?
Sama sekali tidak! Template kami sudah dirancang agar kamu tinggal mengganti teks dan foto tanpa perlu menyentuh struktur kode yang rumit.
Apakah bayarnya per bulan?
Tidak, pembayaran sistem lisensi sekali bayar (one-time payment). Beli sekali, pakai berkali-kali untuk portofoliomu.
Bisa pakai domain sendiri (.com / .id)?
Tentu saja. Setelah source code kamu dapatkan, kamu bisa deploy di mana saja dan menyambungkannya ke domain pribadi.
Apa beda Starter, Pro, dan Premium?
gatau.
Apakah website ini bisa dibuka di HP?
Tentu. Semua template dirancang mobile-first dan sudah diuji dari layar 360px sampai layar desktop besar.
Apakah sistem login ini sistem autentikasi produksi?
Tidak. Halaman login/register saat ini adalah demo prototype berbasis localStorage browser. Data akun hanya tersimpan di perangkat lokal pengguna dan tidak dienkripsi. Sistem ini tidak boleh dianggap sebagai autentikasi produksi. Untuk versi commercial release, autentikasi akan dipindahkan ke backend yang aman.
